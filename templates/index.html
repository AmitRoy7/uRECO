About

uRECO is a web application related to image and gesture recognition developed as a software engieering lab project using Python, TensorFLow HTML, CSS, and JavaScript as well as optional LESS stylesheets for easy customization.

uRECO gives you three awesome features like Face Recognition, Sign Language Detection and Virtual PC controller. So, Create an account by signing in with your email and password and have an experience.!!!
